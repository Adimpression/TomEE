This project is maintained by Adimpression

Driven by David Blevins, TomEE has evolved a lot since its baby stages. Adimpression proudly uses TomEE for most of its servers.

Why?

Because Tomee is pure Tomcat with Java EE capabilities

Because Zetop is open sourced

Because Zepto is well documented

Because TomEE is not messed up. Tomcat is taken clean and configured to be TomEE with minimal interference.

Because TomEE is fully Apache software
